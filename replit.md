# Sonic VIP Store

ستۆرێکی ستاتیک بۆ دابەزاندنی فایلەکانی IPA ی PUBG MOBILE.
دەتوانیت لە ڕێی **پانێڵی ئەدمین** (پارێزراو بە وشەی نهێنی) هەموو وەشانەکان بەڕێوە ببەیت.

## ڕیشەی پرۆژە

- `index.html` — لاپەڕەی سەرەکی (گشتی، بۆ هەموو بەکارهێنەران)
- `style.css` — ستایلی سەرەکی
- `script.js` — لۆجیکی سایتی سەرەکی

- `admin.html` — پانێڵی ئەدمین (پارێزراو بە وشەی نهێنی)
- `admin.css` — ستایلی پانێڵی ئەدمین
- `admin.js` — لۆجیکی پانێڵی ئەدمین

- `data/releases.json` — فایلی ناوەندی هەموو وەشانەکان (سەرچاوەی بنەڕەتی)
- `downloads/` — شوێنی فایلە ڕاستەقینەکانی .ipa (ئیختیاریە)
- `assets/` — وێنە و میدیا

## پانێڵی ئەدمین

URL: `/admin.html`
وشەی نهێنی بنەڕەتی: **`sonic2026`**

> **زۆر گرنگ:** دوای یەکەم چوونە ژوورەوە، وشەی نهێنی بگۆڕە بە دوگمەی «گۆڕینی وشەی نهێنی»
> یان لە ناو فایلی `admin.js` لە دێڕی ١٠، بەهای `DEFAULT_PASSWORD` بگۆڕە.

### چی دەتوانیت بکەیت لە پانێڵدا
- زیادکردن، گۆڕین، یان سڕینەوەی هەموو وەشانێک
- گۆڕینی ستاتیستیکی سەرەکی (دابەزاندن، بەکارهێنەر، هەڵسەنگاندن…)
- **داگرتنی هەموو فایلەکان** بە یەک فایلی JSON
- **هاوردنی** فایلێکی JSON (بۆ گەڕاندنەوە یان گواستنەوە)
- گۆڕینی وشەی نهێنی

### چۆن گۆڕانکارییەکان بۆ هەموو سەردانیکەران ببینرێت

پانێڵی ئەدمین گۆڕانکارییەکان لە **localStorage**ی براوزەری خۆتدا پاشەکەوت دەکات.
بۆ ئەوەی هەموو سەردانیکەرانی سایتەکە گۆڕانکارییەکان ببینن:

1. لە پانێڵی ئەدمین، دوگمەی **«داگرتنی هەموو فایلەکان (JSON)»** کلیک بکە
2. فایلی داگیراو دانێ بە جێی فایلی `data/releases.json`
3. سایت ڕەش بکەرەوە (republish)

دواتر هەموو کەس وەشانە نوێیەکان دەبینن.

## چۆن فایلێکی IPA زیاد بکەیت

### ڕێگای ١ — بە پانێڵی ئەدمین (پێشنیار دەکرێت)
١. بڕۆ بۆ `/admin.html` و وشەی نهێنی بنووسە
٢. دوگمەی «زیادکردنی وەشانێکی نوێ» کلیک بکە
٣. زانیارییەکان پڕ بکەرەوە، گرنگترین خانە **«لینکی دابەزاندن»**ـە
٤. لینکەکە دەتوانێت یەکێک بێت لەمانە:
   - لینکی تەلیگرام (`https://t.me/sonic_vip01/123`)
   - لینکی MediaFire، Google Drive، Mega، هتد
   - یان ڕێچکەی ناوخۆیی (`downloads/PUBG_KR.ipa`) ئەگەر فایلت بارکردووە بۆ پرۆژە
٥. پاشەکەوتی بکە — یەکسەر لە سایتەکە دەردەکەوێت

### ڕێگای ٢ — بە دەستی (ئاسانترە بۆ بارکردنی فایلی ڕاستەقینە)
١. فایلی `.ipa` دانێ لەناو فۆڵدەری `downloads/`
٢. تۆمارێک زیاد بکە بۆ `data/releases.json`:
```json
{
  "id": "pubg-kr-350",
  "title": "PUBG MOBILE KR",
  "subtitle": "وەشانی کۆریا",
  "region": "KR",
  "version": "3.5.0",
  "size": "1.2 GB",
  "date": "2026-05-01",
  "ios": "iOS 13+",
  "featured": true,
  "description": "...",
  "file": "downloads/PUBG_KR_3.5.0.ipa",
  "tags": ["نوێ"]
}
```

ناوی regionەکانی پشتیوانیکراو: `KR`, `VNG`, `TW`, `GLOBAL`, `BGMI`, `LITE`.
